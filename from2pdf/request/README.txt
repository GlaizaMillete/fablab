Service Request Form Application Installation Guide

Prerequisites:
1. Python 3.8 or higher (Download from https://www.python.org/downloads/)
   - During installation, make sure to check "Add Python to PATH"

Installation Steps:
1. Extract all files from the zip archive
2. Double-click on "install_and_run.bat"
3. Wait for the installation to complete
4. The executable will be created in the 'dist' folder

Running the Application:
- Go to the 'dist' folder and run servicereq.exe

Note: If you encounter any issues:
- Make sure Python is installed and added to PATH
- Try running the batch file as administrator
- Check that all files are in the same folder

Files included in this package:
- servicereq.py (Source code)
- requirements.txt (Required Python packages)
- request.pdf (Template form)
- install_and_run.bat (Installation script)
